# Pagina-web-games 🎮

## O Projeto 💻:
O objetivo do projeto foi fazer uma página web para uma empresa de jogos fictícia. O site teria que ter home, projetos, parceiros, Faq e contato

## Linguagens ✏️:
As linguagens utilizadas foram: HTML, CSS e Framework 

## Arquivos 📲:
Os arquivos html e css juntos criam o site completo com divisões e imagens. No arquivo "imagens", contém algumas fotos utilizadas no projeto e o site entre si funcionando. Por fim, o arquivo "interação com ia" contém os registros de comunicação com a inteligência artificial ao longo do projeto.

## Como executar 🧑‍💻
Baixe o arquivo "index.html" e "estilo css". Abra o arquivo do portal da pasta em seu computador.
